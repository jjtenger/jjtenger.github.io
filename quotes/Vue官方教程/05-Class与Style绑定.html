  • 05-CLASS与STYLE绑定

在将 v-bind 用于 class 和 style 时，Vue.js 做了专门的增强。表达式结果的类型除了字符串之外，还可以是对象或数组。

  • 绑定 HTML Class
    • 对象语法
      传给 v-bind:class 一个数据对象，key 为 class 名，value 为真值，该 class 存在与否将取决于 value 值。
      此外，v-bind:class 指令也可以与普通的 class 属性共存。
      绑定的数据对象不必内联定义在模板中，也可以绑定一个 data 中定义好的属性对象，还可以绑定一个计算属性。
    • 数组语法
      可以把一个数组传给 v-bind:class，数组中各元素应为 data 中的绑定属性，对应属性值决定了 class 名称。
      使用三元表达式切换列表中的 class：[isActive ? activeClass : '', errorClass]。
      数组语法中也可以使用对象语法：[{ active: isActive }, errorClass]。
    • 用在组件上
      当在一个自定义组件上使用 class 属性时，这些类将被添加到该组件的根元素上面。这个元素上已经存在的类不会被覆盖。
  • 绑定内联样式
    • 对象语法
      可以在 v-bind:style 模版中定义内联 CSS 对象，key 为 CSS 属性名；value 为 CSS 属性值，可以绑定至 data 属性中。
      更好的方式是直接绑定一个 data 中定义好的样式对象。同样的，对象语法常常结合返回对象的计算属性使用。
