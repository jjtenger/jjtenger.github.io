  • 06-条件渲染
  • v-if
    条件性渲染标签内容（只在指令表达式返回 truthy 值时被渲染）。
    • 在 template 元素上使用 v-if 条件渲染分组
      把 <template> 元素当做不可见的包裹元素，在上面使用 v-if 。
    • v-else
      v-if 的“ else 块”，必须紧跟带 v-if 或者 v-else-if 的元素后面。
    • v-else-if
      v-if 的“ else-if 块”，可以连续使用，也必须紧跟带 v-if 或者 v-else-if 的元素后面。
    • 用 key 管理可复用的元素
      Vue 通常会复用已有元素，因此不会清除用户已输入的内容。
      使用 key 属性表示两个元素是完全独立的（不可复用）。
  • v-show
    v-show 只是简单地切换元素的 CSS 属性 display ，而不是重新渲染。
  • v-if vs v-show
    v-if 是“真正”的条件渲染，也是惰性的（只在条件为真时渲染）。
    一般来说，v-if 有更高的切换开销，而 v-show 有更高的初始渲染开销。
  • v-if 与 v-for 一起使用
    不推荐同时使用 v-if 和 v-for 。
