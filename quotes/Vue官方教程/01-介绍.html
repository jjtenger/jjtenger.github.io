  • Vue.js 是什么
    用于构建用户界面的渐进式框架，分层设计，核心库只关注视图层。
  • 起步
    安装：<script>引入，开发环境版本和生产环境版本。
  • 声明式渲染
    { { } }：文本插值。
    v-bind 指令：元素特性绑定。
  • 条件与循环
    v-if 指令：控制切换一个元素。（不仅可以把数据绑定到 DOM 文本或特性，还可以绑定到 DOM 结构）。
    v-for 指令：绑定数组的数据来循环渲染一个项目列表。
  • 处理用户输入
    v-on 指令：添加一个事件监听器，通过它调用 Vue.methods 中的方法。所有的 DOM 操作都由 Vue 来处理，无需用户处理。
    v-model 指令：实现表单输入和应用状态之间的双向绑定。
  • 组件化应用构建
    组件：小型、独立、可重复。应用界面抽象为一个组件树。
    注册与使用组件：Vue.component。
    从父作用域将数据传到子组件，解耦子单元与父单元。
