变量类型

虽然JavaScript的变量不具有数据类型，但从概念上，JavaScript变量可以分为基本数据类型变量和引用类型变量。

  • 引用类型变量：保存对象的引用。
  • 基本数据类型变量：直接保存有数值等类型的数据的值。

变量的赋值

基本数据类型变量的赋值是直接数据值的传递；而引用类型变量的赋值是将对象的引用（一种用于指示对象位置的标记）赋值给变量，归根到底还是值（引用地址）传递。

变量命名

变量名可以任意取名，但要遵循命名规则:

  • 变量必须使用字母、下划线(_)或者美元符($)开始。
  • 然后可以使用任意多个英文字母、数字、下划线(_)或者美元符($)组成。
  • 不能使用JavaScript关键词与JavaScript保留字。

在JS中区分大小写，如变量mychar与myChar是不一样的，表示是两个变量。
