  • 08-控制语句

控制语句包括条件分支、循环、跳转（包括异常处理）这3类。如果没有这样的控制语句，JavaScript在理论上，是按照源代码上所写的代码顺序从上至下地执行。这种执行方式被称为“顺序执行”。

switch-case语句

在JavaScript中，switch的括号内可以写任意类型的表达式，case标签中也可以写任意的表达式。但在Java的case标签中，则只能使用在编译时就能够获得结果的常量表达式。

switch语句会把其在switch之后的括号内的表达式，与case标签中所写的各个表达式，依次通过相等运算符（===，不会对数据类型进行转换）进行比较。

注意通过break语句跳出switch语句，没有break语句的switch语句，将不会在执行完其中某一段case之后就结束整个switch语句。

for语句

一个典型的for语句，将会在初始化表达式中对循环变量进行初始化(初始化表达式只执行一次)，然后在条件表达式中检查循环变量的值，如果条件为真，则执行循环语句，最后再在更新表达式中对循环变量进行更新。之后条件表达式将被再次求值……

  • for in语句可以用来枚举数列的元素（但不推荐这么做）。
  • 虽然可能绝大多数时候都能够按照预期的顺序输出，不过for in语句本身并不保证枚举的顺序，所以仍然不应该对此过分依赖。

  • for in语句还可以枚举由原型继承而来的属性。

for each in语句

for each in语句在ECMAScript中并不存在，是JavaScript自有的增强功能。

和for in不同，for each in语句并不是把属性名赋值给变量，而是将属性值赋值给它。其它用法与for in相同。

while语句

和for循环有相同功能的还有while循环, while循环重复执行一段代码，直到某个条件不再满足。

和if-else语句一样，在条件表达式位置所写的表达式的值将会被转换为布尔型。

do-while语句

while语句与do-while语句的差别仅仅在于，是首先对条件表达式进行求值，还是首先执行语句。

break与Continue

break退出循环

使用了break语句的情况下，不会再一次对条件表达式进行求值。如果是for语句的情况，则不会对更新表达式求值。

在嵌套循环的内层使用break语句的话，仅仅会跳出内层的循环。

Continue继续循环

跳过在此之后本次循环内尚未执行的语句，而返回至循环的条件表达式进行求值。如果是for语句的情况，则是返回至更新表达式处求值后再对条件表达式进行求值。
