特征组合 (Feature Crosses)：对非线性规律进行编码

特征组合是指通过将两个或多个输入特征相乘来对特征空间中的非线性规律进行编码的合成特征。我们像处理任何其他特征一样来处理这个新建的特征。
在使用扩展的线性模型时辅以特征组合一直都是训练大规模数据集的有效方法。

特征组合的种类

[A X B]、[A x B x C x D x E]、[A x A]

组合独热矢量

在实践中很少会组合连续特征。经常组合独热特征矢量，将独热特征矢量的特征组合视为逻辑连接（AND）。
