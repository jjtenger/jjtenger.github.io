2.变量名

在Java中，变量名以下划线、字母、$符号开头，并且后跟下划线

字母、$符号以及数字。总之，Java中的变量名不能以数字开头。

3.注释

注释是给人看的，不是给计算机看的。Java 中共有3种类型的注释

1）单行注释：以//开头，//后面所有内容均当作注释内容处理。

2）多行注释：以/开头，以/结束，中间所有内容均被当作注释处理。多行注释来源于C/C++。注意多行注释不能嵌套。

3）另一种多行注释：以/*开头，以/结尾，用于产生Java Doc帮组文档。

4.数据类型

Java中的数据类型分为两大类：

原生数据类型（Primitive Data Type）

Java中的原生数据类型共有8种

1）整型：使用int表示（32位，FFFF FFFF）。

2）字节型：使用byte表示（表示-128~127之间的256个整数）（8位，FF）。

3）短整型：使用short表示（16位，双字节，FFFF）。

4）长整型：使用long表示（64位）。

5）单精度浮点型：使用float表示。

6）双精度浮点型：使用double表示（表示的数据范围比单精度浮点型大）。

7）字符型：使用char表示（character的缩写）。所谓字符，就是单个的字符表示。

比如字母a，或者中文张，，外面用单引号包围。比如：char a = ‘B’；char b = ‘张’；

8）布尔类型，使用boolean表示。布尔类型只有两种可能值，分别是true与false。

数据类型使用陷阱

1）Java中所有浮点类型数值默认情况下都是double类型。不能直接将double类型的值赋给

float类型的变量。

2）可以将表示范围小的值赋给表示范围大的变量，不能将表示范围大的值赋给表示范围小的变量

3）如何将表示范围大的值赋给表示范围小的变量：强制类型转换。语法：1）

类型 变量名 = （类型）变量值；2）double i=8.45f; //double类型转换为float类型

4）变量在使用前必须定义，声明其类型并赋值才能使用，并且变量只能定义一次。

引用类型（reference type）

引用类型是用在对象上的。一个对象可以被多个引用所指向。但同一时刻，每个引用只能指向唯一的一个对象。如果一个对象被多个引用所指向，那么无论哪个引用对对象的属性进行了修改，都会反映到其他的引用当中。（引用变量里寄存的是堆中对象的地址，而普通变量里寄存的是变量的值）

引用类型只能指向其相同类型的对象，比如People类型的引用能指向People类型的对象，但不能指向Student类型的变量（不考虑继承关系）。

5.数据单位

关于计算机系统中的数据表示

1）位：bit（只有0，1两种状态），是计算机系统中的最小数据表示单位。

2）字节：byte，1 byte = 8 bit 。

3）1KB = 1024Byte。1MB = 1024KB. 1GB = 1024MB。
