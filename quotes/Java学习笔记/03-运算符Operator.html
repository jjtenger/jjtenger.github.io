1.算术运算符

整数除法的结果得到的是整数商（舍去余数的结果），非四舍五入的结果。

当有若干个变量参与运算时，结果类型取决于这些变量中表示范围最大的那个变量类型。比如参与运算的变量中，有整形int，有双精度浮点型doule，有短整型short，那么最后的结果类型是double。如果结果值类型与等号左边赋给的变量类型不一致，有可能导致语法错误：损失进度。

取模运算符：求取余数，使用%表示。取模的规律：取模的结果符号永远与被除数的符号相同（被除数/除数）。

关于变量的自增与自减运算

  • 关于int b = a++，作用是先将a的值先赋给b，然后再让a自增1。
  • 关于int b = ++a，作用是先将a自增1，然后再让a的值赋给b。

2.关系运算符

大于（>）、小于（ <）、等于（==）、不等于（!=）、大于等于（>=）、小于等于（<=），

关系运算的结果都是boolean值。

相等性比较（==）

  • 对于原生数据类型来说，比较的是左右两边的值是否相等。
  • 对于引用类型来说，比较左右两边的引用是否指向同一个对象，或者说左右两边的引用地址是否相同。

3.逻辑运算符

重点讲解两个：逻辑运算符本身也返回一个boolean值。

  • 逻辑与：使用&&表示。只有当两个操作数都为真的时候，结果才为真。其余都为假。
  • 逻辑或：使用||表示，只有当两个操作数都为假时都为假，其余都为真。

关于逻辑运算符的短路特性：

  • 逻辑与：如果第一个操作数为false，那么结果肯定是false，所以在这种情况下，就不会执行逻辑与后面第二个操作数中的运算了，即发生了短路。
  • 逻辑或：如果第一个操作数为true，那么结果肯定是true，此时也会发生短路。

4.条件运算符（三元表达式）

其形式为：

a为true时，将b赋值给d；a为false时，将c值赋值给d。
