其它Maven构建命令

注意：先进入项目目录后再操作！

命令 说明
mvn clean 清除原来的编译结果
mvn compile 编译
mvn test 运行测试代码：mvn test -Dtest=类名//单独运行测试类
mvn package 打包项目：mvn package -Dmanven.test.skip=true//打包时不执行测试
mvn install 将项目打包并安装到本地仓库
mvn deploy 发布到本地仓库或者服务器
