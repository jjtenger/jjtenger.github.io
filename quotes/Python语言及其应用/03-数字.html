Python语言及其应用-第2章数字、字符串和变量（P4）

Python 本身支持整数（ 比如 5 和 1000000000） 以及浮点数（ 比如 3.1416、14.99 和 1.87e4）。你可以对这些数字进行 +、-、*、/、//（整除）、%（模，求余）、** （幂）等计算。

整数与浮点数

整数全部由数字组成，而浮点数（在 Python 里称为 float）包含非数字的小数点。
对于整数，数字零（0）不能作为前缀。

浮点除法与整数除法

Python语言及其应用-第2章数字、字符串和变量（P6）

Python 里有两种除法：

  • / 用来执行浮点除法（十进制小数）
  • // 用来执行整数除法（整除）
    与其他语言不同，在 Python 中即使运算对象是两个整数，使用 / 仍会得到浮点型的结果。
    使用整除运算得到的是一个整数，余数会被截去。
    如果除数为 0，任何一种除法运算都会产生 Python 异常。

基数

Python语言及其应用-第2章数字、字符串和变量（P9）

在 Python 中，整数默认使用十进制数（以 10 为底），除非你在数字前添加前缀，显式地指定使用其他基数（base）。
基数指的是在必须进位前可以使用的数字的最大数量。以 2 为底（二进制）时，可以使用的数字只有 0 和 1。这里的 0 和十进制的 0 代表的意义相同，1 和十进制的 1 所代表的意义也相同。然而以 2 为底时，1 与 1 相加得到的将是 10（1 个二加 0 个一）。
在 Python 中，除十进制外你还可以使用其他三种进制的数字：

  • 0b 或 0B 代表二进制（以 2 为底）
  • 0o 或 0O 代表八进制（以 8 为底）
  • 0x 或 0X 代表十六进制（以 16 为底）
    Python 解释器会打印出它们对应的十进制整数。

转换为整型：int()

Python语言及其应用-第2章数字、字符串和变量（P10-11）

我们可以方便地使用 int() 函数将其他的 Python 数据类型转换为整型。它会保留传入数据的整数部分并舍去小数部分。
Python 里最简单的数据类型是布尔型，它只有两个可选值：True 和 False。当转换为整数时它们分别代表 1 和 0。
当将浮点数转换为整数时，所有小数点后面的部分会被舍去。
也可以将仅包含数字和正负号的字符串转换为整数。但如果你试图将一个与数字无关的类型转化为整数，会得到一个异常。而且，int() 可以接受浮点数或由数字组成的字符串，但无法接受包含小数点或指数的字符串。
如果混合使用多种不同的数字类型进行计算，Python 会自动地进行类型转换。
与整数或浮点数混合使用时，布尔型的 False 会被当作 0 或 0.0，Ture 会被当作 1 或 1.0。

一个int型有多大

Python语言及其应用-第2章数字、字符串和变量（P11）

在 Python2 里，一个 int 型包含 32 位，可以存储从 -2147483648 到 2147483647 的整数。一个 long 型会占用更多的空间：64 位，可以存储从 -9223372036854775808 到 9223372036854775807 的整数。
到了 Python 3，long 类型已不复存在，而 int 类型变为可以存储任意大小的整数，甚至超过 64 位。

转换为浮点型：float()

Python语言及其应用-第2章数字、字符串和变量（P12）

使用 float() 函数可以将其他数字类型转换为浮点型。与之前一样，布尔型在计算中等价于 1.0 和 0.0。
此外，也可以将包含有效浮点数（数字、正负号、小数点、指数及指数的前缀 e）的字符串转换为真正的浮点型数字。
