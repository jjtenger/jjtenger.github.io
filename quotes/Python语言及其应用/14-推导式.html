列表推导式

Python语言及其应用-第4章代码结构（P12-14）

更像 Python 风格的创建列表方式是使用列表推导。最简单的形式如下所示：
[ expression for item in iterable ]
其中，expression 为列表生成值，可以为表达式；item 为 iterable 迭代结果。返回的最终结果为所有生成值组成的列表。

列表推导也可以像下面的例子加上条件表达式：
[expression for item in iterable if condition]

字典推导式

Python语言及其应用-第4章代码结构（P14）

除了列表，字典也有自己的推导式。最简单的例子就像：
{ key_expression : value_expression for expression in iterable }
类似于列表推导，字典推导也有 if 条件判断以及多个 for 循环迭代语句。

其实，圆括号之间的是生成器推导式，它返回的是一个生成器对象。

它是将数据传给迭代器的一种方式。你可以直接对生成器对象进行迭代，或者通过对一个生成器的推导式调用 list() 函数，使它类似于列表推导式。
一个生成器只能运行一次。列表、集合、字符串和字典都存储在内存中，但 是生成器仅在运行中产生值，不会被存下来，所以不能重新使用或者备份一个生成器。
